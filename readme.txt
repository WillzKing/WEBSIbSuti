Утсановить зависимости придётся в ручную. Список зависимостей:

pip install django
pip install djangorestframework
pip install djangorestframework-simplejwt
pip install pillow
pip install django-cors-headers
pip install psycopg2-binary

Это минимум. Так же понадобится Python, желательно версии 3.10+, node, npm и psql.
